# Student-Progress-Tracking-Recommendation-System
This project uses Python and data analytics to track student progress and identify skill gaps. It offers personalized recommendations to improve learning, highlights students needing support, and spots challenging skills. The complete solution includes data analysis, visualization, and CSV reports for educational dashboards.

## Features
- Analyzes student performance across various skills
- Generates CSV recommendations for weak skills
- Highlights most challenging skills and students with the largest skill gaps

## How to Run
1. Upload your dataset (CSV) in Google Colab.
2. Run the notebook step by step to analyze and generate recommendations.
3. Download the produced output CSV files.
